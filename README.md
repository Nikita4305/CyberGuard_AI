# CyberGuard AI

## Multi-Agent Cybersecurity Incident Response Assistant

CyberGuard AI is a multi-agent cybersecurity incident response assistant developed as part of the Google × Kaggle 5-Day AI Agents: Intensive Vibe Coding Capstone Project.

The system helps security analysts and organizations quickly identify, analyze, prioritize, and respond to cybersecurity incidents using multiple AI agents.

## Features

* SSH Brute Force Detection
* Phishing Domain Detection
* Port Scan Detection
* Malware Hash Analysis
* DDoS Detection
* Ransomware Detection
* Risk Assessment
* Automated Incident Response
* Incident Report Generation

## Architecture

The system consists of multiple cooperating AI agents:

* Coordinator Agent
* Log Analysis Agent
* Threat Intelligence Agent
* Risk Assessment Agent
* Response Agent
* Report Generation Agent
